# 5주차 과제

## 목표

* 최대 힙,최소 힙 중 1가지 구현
* 다음의 연산을 수행

## 추가 목표
* 힙을 이용하여 정렬 (정렬할 구간은 arr[1...n])

---
1. init 연산: 힙 초기화

2. isEmpty 연산: 힙이 비었는 지 확인한다.
    * 힙이 비었다면 1, 아니면 0 을 반환한다.

3. push 연산: 힙에 임의의 데이터를 삽입한다.

4. peek 연산: 힙의 루트 노드에 저장된 값을 반환한다.

5. pop 연산: 힙의 루트 노드를 제거한다. 그 후, 다시 힙 구조를 이루도록 한다.
